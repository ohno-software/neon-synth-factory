# Neon Synth Factory - Design System

The visual identity for Neon Synth Factory centers on a high-contrast, "Cyber-Retro" aesthetic with glow effects, sharp vector lines, and a dark workspace.

## Color Palette

### Backgrounds
- **Deep Space:** `#0A0A0E` (Main window background)
- **Panel Dark:** `#121218` (Section background)
- **Panel Light:** `#1C1C26` (Lifted surface)

### Brand Accents (The "Neon")
- **Envelope Green:** `#00FF00` (Max brightness for envelopes)
- **Oscillator Cyan:** `#00AAFF` (Cool blue for waveforms)
- **Filter Teal:** `#00FFAA` (Bright cyan for frequency controls)
- **Modulation Orange:** `#FFAA00` (Warm amber for routing/LFOs/Envelopes)
- **Volcanic Orange:** `#FF4D00` (High-energy output/amp color)
- **Pan Pink:** `#FF00AA` (High saturation magenta for sequencing/arps)
- **Sky Blue:** `#0088FF` (Deep blue for effects)

### Status & Feedback
- **Value Indicator:** `#FFFFFF` (Solid white circle, 10px)
- **Active State Glow:** 30% opacity of accent colors.
- **Empty Track/Bar Background:** `#404040` (Dark gray for un-filled bars)
- **Text Primary:** `#FFFFFF` (Pure white for labels)
- **Text Value:** `#CCCCCC` (Light gray for readouts)

## Typography
- **Primary Font:** Sans-serif, high legibility (e.g., Roboto or Inter).
- **Secondary/Display Font:** **Pseudo 7-Segment Font** for value readouts (e.g., "DSEG7" style or similar digital display font).
- **Sizes:**
  - **Parameter Labels:** 14px Bold, **ALWAYS UPPERCASE**
  - **Value Readouts (7-Segment):** 16px (increased for legibility of segments)
  - **Section Titles:** 16px Semi-bold

## Visual Language
- **Card Corner Radius:** 5px
- **Bar Corner Radius:** 3px
- **Outline Weight:** 1.0px for standard, 2.0px for focus/active.
- **7-Segment Style:** Values should appear "inset" or glow like a classic hardware LED display.
- **Layout Pattern:** Horizontal "Parameter Cards" (approx. 1/4 panel width). Every single UI interaction happens within a card.
