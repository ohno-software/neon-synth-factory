# Module Parameter Definitions

This document lists the parameters available for each module type in the Neon Synth Factory.

## Modules

### [Oscillator]
*   **Waveform**: dropdown, neon-jr/waves-1, index 0
*   **Symmetry**: slider, 0.0, 1.0, 0.5
*   **Detune**: slider, -100.0, 100.0, 0.0
*   **Transp**: slider, -24.0, 24.0, 0.0
*   **Phase**: slider, 0.0, 360.0, 0.0
*   **KeySync**: toggle, OFF, ON, ON
*   **Volume**: slider, 0.0, 1.0, 0.8
*   **Pan**: slider, -1.0, 1.0, 0.0
*   **Drive**: slider, 0.0, 1.0, 0.0
*   **BitRedux**: slider, 0.0, 1.0, 0.0
*   **Fold**: slider, 0.0, 1.0, 0.0
*   **Unison**: slider, 0.0, 8.0, 0.0

### [Ladder Filter]
*   **Cutoff**: slider, 20.0, 20000.0, 20000.0
*   **Res**: slider, 0.0, 1.0, 0.0
*   **Drive**: slider, 0.0, 10.0, 0.0
*   **Slope**: toggle, 12dB, 24dB, 24dB
*   **KeyTrack**: slider, 0.0, 1.0, 0.5

### [Amp Output]
*   **Level**: slider, 0.0, 1.0, 0.8
*   **Pan**: slider, -1.0, 1.0, 0.0
*   **Spread**: slider, 0.0, 1.0, 0.0
*   **Velocity**: slider, 0.0, 1.0, 0.5

### [DAHDSR Envelope]
*   **Delay**: slider, 0.0, 10000.0, 0.0
*   **Attack**: slider, 0.0, 10000.0, 100.0
*   **Hold**: slider, 0.0, 10000.0, 0.0
*   **Decay**: slider, 0.0, 10000.0, 500.0
*   **Sustain**: slider, 0.0, 1.0, 0.7
*   **Release**: slider, 0.0, 10000.0, 1000.0
*   **Amount**: slider, -1.0, 1.0, 0.5 (Optional focus parameter)

### [ADSR Envelope]
*   **Attack**: slider, 0.0, 10000.0, 100.0
*   **Decay**: slider, 0.0, 10000.0, 500.0
*   **Sustain**: slider, 0.0, 1.0, 0.7
*   **Release**: slider, 0.0, 10000.0, 1000.0
